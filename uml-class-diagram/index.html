    Class Diagram

    A class diagram is a structural UML diagram that shows the system's classes, their attributes, methods, and the relationships between them (inheritance, association, aggregation, composition, dependency).

    Diagram

    Key Concepts

    Relationship Notation Meaning
    Inheritance Solid line + hollow arrow "is-a"
    Association Solid line "has a reference to"
    Aggregation Solid line + hollow diamond "has a" (weak ownership)
    Composition Solid line + filled diamond "owns a" (strong ownership)
    Dependency Dashed line + arrow "uses temporarily"
    Realization Dashed line + hollow arrow "implements interface"
